# Fake News Detection using Neural Network

## Project Description
This project uses a Neural Network to classify news headlines as Fake or Real.

## Model Details
- Architecture: Feedforward Neural Network (256 → 128 → 64 → Output)
- Activation Functions: ReLU (hidden), Sigmoid (output)
- Epochs: 15

## Performance
- Accuracy: 99.4 %

## Technologies Used
- Python
- TensorFlow / Keras
- Scikit-learn
- Pandas

## Example Predictions
- "Scientists confirm water on Mars." → Real News
- "Aliens have taken over the government." → Fake News
